# go-webterm: Terminal over the Browser

go-webterm is a [xterm.js](https://xtermjs.org/) Go backend for learning (and fun!) purposes

## Requirements

- Go >= 1.20

## Build and Usage

Running `make build` will generate the binary `bin/webterm`

```bash
bin/webterm
Hello, World!
```
